# 产品管理操作手册

每次新增产品只需修改本地 2 个地方，然后推送到 GitHub，网站自动更新。

---

## 第一步：准备产品图片

1. 把产品图片放到文件夹：
   ```
   C:\Users\0\WorkBuddy\2026-05-21-18-22-29\mementopaws\public\images\
   ```
2. 图片建议：JPG 格式，宽度 1200px 以上，文件大小 < 500KB
3. 记下文件名，比如 `urn-new-product.jpg`

---

## 第二步：添加产品数据

用记事本或 VSCode 打开文件：
```
C:\Users\0\WorkBuddy\2026-05-21-18-22-29\mementopaws\src\lib\data.ts
```

找到 `products` 数组（文件开头第 3 行附近），在最后一个产品 `}` 后面加逗号，然后粘贴新产品的数据模板。

### 数据模板（复制后修改括号里的内容）

```typescript
  {
    id: '产品ID',                              // 如 'urn-03'
    slug: '产品网址-slug',                      // 如 'new-ceramic-urn'
    name: '产品名称',                           // 如 'Hand-Painted Ceramic Urn'
    category: 'urns',                          // 选择：urns / felt-replicas / candles / cards / photo-cards / gift-boxes / remembrance-kits
    tagline: '一句话打动人的描述',                // 如 'A vessel of quiet remembrance'
    description: '简短描述（产品卡片用）',
    longDescription: '长篇情感描述（产品详情页用，可写 2-4 句）',
    price: 480,                                // 价格（数字）
    materials: ['材质1', '材质2', '材质3'],       // 材质列表
    dimensions: '20cm x 15cm x 15cm',           // 尺寸
    craftingTime: '10-14 business days',        // 制作时间
    features: [                                 // 产品特性列表
      '特性1',
      '特性2',
      '特性3',
      '特性4',
      '特性5',
      '特性6',
    ],
    images: ['/images/你的图片文件名.jpg'],       // 和第一步放进去的文件名一致
    featured: false,                            // true=首页推荐  false=不推荐
    isNew: true,                                // true=显示New标签  false=不显示
  },
```

### ⚠️ 注意事项

- 每行末尾要有半角逗号 `,`
- 最后一个产品的大括号 `}` 后面要加逗号，再粘贴新产品
- 数组最后一个产品不要加逗号（但加了也能跑）
- `images` 里的路径必须以 `/images/` 开头

---

## 第三步：推送到 GitHub

打开终端（Win + R → `cmd`），逐行输入以下命令：

```bash
cd C:\Users\0\WorkBuddy\2026-05-21-18-22-29\mementopaws

git add -A

git commit -m "新增产品：你的产品名称"

git push
```

每行输完按回车。如果弹出 GitHub 登录窗口，点浏览器授权。

---

## 完成

推送后等待 2-3 分钟，打开 `https://mementopaws.pages.dev` 刷新，新产品就出现了。

以后每次加产品，重复这三步即可：
1. 放图片到 `public/images/`
2. 复制数据模板到 `src/lib/data.ts`
3. 终端执行 `git add -A && git commit -m "新增产品" && git push`
